About the project

This project gave me the an opportunity, as learning developer, to test and expand my skills when it come to Angular development. It was designed by NTT DATA in partnership with Start2Impact and will be reviewed for feedback.

The requiremts

The requirements are to develop an application based on the APIs provided by Go REST. It's particularly important to study the API documentation in order to make API calls, understand the input parameters, response structure, and HTTP response codes.

The application should allows managing users and viewing/editing their posts and the functionalities offered by the application must be protected by bearer token authentication. The implementation of a server-side component is not required.

The Specs

After authentication, the user should access a section where they can see the list of users, which includes search by name option. It should be possible to create or delete a new user.

The user should be able to open a user's profile, where their details are displayed, view their posts, and leave comments.

There should be a second functionality where the user can view all posts, perform searches, and once they find the post of interest, view and add comments. In this section, the user can also create new posts.
